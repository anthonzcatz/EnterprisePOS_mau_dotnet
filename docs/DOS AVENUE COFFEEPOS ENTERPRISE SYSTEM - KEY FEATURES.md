DOS AVENUE COFFEEPOS ENTERPRISE SYSTEM - KEY FEATURES
1. AUTHENTICATION & SECURITY
Description

Responsible for securing the system, managing user access, authentication, authorization, and activity monitoring.

Features

• User registration and management
• Secure login using JWT + Refresh Tokens
• Role-based access control (Admin, Manager, Cashier, Owner)
• Permission-based access control
• Password hashing and encryption
• Session management
• Device session tracking
• Login history monitoring
• Audit trail for all critical actions

2. DASHBOARD
Description

Provides a real-time overview of business operations, sales performance, inventory status, receivables, and key business metrics.

Features

• Today's, Weekly, Monthly, and Yearly Sales
• Sales Trend Charts
• Top Selling Products
• Low Stock Alerts
• Outstanding Receivables Summary
• Cash Drawer Summary
• Payment Method Breakdown
• Recent Activities Feed

3. PRODUCT MANAGEMENT
Description

Centralized module for managing products, categories, pricing, variants, barcodes, and product-related configurations.

Features

• Product Categories Management
• Product Registration and Maintenance
• Product Variants (Small, Medium, Large)
• Product Add-ons Management
• Product Images
• Product Barcodes
• Multiple Price Levels
• Retail, Recipe, and Service Product Types

4. RECIPE & BOM MANAGEMENT
Description

Allows the system to define product recipes and automatically deduct ingredients from inventory whenever recipe-based products are sold.

Features

• Recipe (Bill of Materials) Setup
• Ingredient Management
• Automatic Ingredient Deduction on Sales
• Recipe Cost Calculation
• Profit Margin Computation
• Ingredient Usage Tracking

Example:

Cafe Latte

Coffee Beans - 20g
Milk - 200ml
Sugar - 10g
Cup - 1pc

When a Latte is sold, inventory is automatically deducted.

5. UNIT OF MEASURE & CONVERSION
Description

Handles multiple units of measure and automatic conversion between purchasing units and selling units.

Features

• Unit Management
• Product Unit Conversions
• Multi-Pack Support
• Purchase by Case, Sell by Piece Support
• Automatic Unit Conversion
• Cost Conversion Calculation

Example:

Purchase:
10 Cases of Coke

1 Case = 24 Bottles

Inventory:
240 Bottles
6. INVENTORY MANAGEMENT
Description

Tracks inventory movement, stock levels, stock adjustments, physical counts, and inventory valuation.

Features

• Real-Time Inventory Monitoring
• Stock In / Stock Out
• Stock Adjustments
• Inventory Variance Tracking
• Batch Tracking
• Expiration Tracking
• Spoilage Management
• Inventory Audit Reports

7. PURCHASING MANAGEMENT
Description

Manages supplier transactions, purchase orders, receiving operations, and procurement history.

Features

• Supplier Management
• Purchase Orders
• Purchase Approval Workflow
• Goods Receiving
• Partial Receiving Support
• Supplier Cost Analysis

8. SALES & POS MANAGEMENT
Description

The primary cashier module used to process customer transactions quickly and efficiently.

Features

• Touchscreen-Friendly POS Interface
• Barcode Scanning
• Product Search
• Add-ons and Variants Support
• Discounts
• Hold and Resume Transactions
• Receipt Printing
• Offline Sales Processing

9. PAYMENT MANAGEMENT
Description

Handles all payment transactions including cash, digital wallets, bank payments, split payments, and partial payments.

Features

• Cash Payments
• GCash Payments
• Maya Payments
• Credit/Debit Cards
• Split Payments
• Partial Payments
• Payment History

10. CUSTOMER MANAGEMENT
Description

Maintains customer information, transaction history, loyalty rewards, and store credit balances.

Features

• Customer Profiles
• Customer Notes
• Purchase History
• Loyalty Program
• Customer Wallet
• Customer Reports

11. RECEIVABLES MANAGEMENT
Description

Manages customer credit transactions, payment collections, balances, and overdue accounts.

Features

• Charge Accounts
• Credit Sales
• Partial Payments
• Due Date Monitoring
• Aging Reports
• Write-Off Management

12. CASHIERING MANAGEMENT
Description

Tracks cashier shifts, cash drawer activities, cash reconciliation, and daily balancing.

Features

• Cash Drawer Control
• Open/Close Shift
• Cash In / Cash Out
• Cash Reconciliation
• Variance Monitoring

13. REPORTING & ANALYTICS
Description

Provides business intelligence, operational reports, financial summaries, and performance analytics.

Features

• Sales Reports
• Inventory Reports
• Product Performance Reports
• Customer Reports
• Receivable Reports
• Profit Analysis Reports
• Audit Reports

14. BIR COMPLIANCE MODULE
Description

Ensures compliance with Philippine BIR requirements for receipts, transaction numbering, reports, and audit readiness.

Features

• PTU Management
• Accreditation Information
• Receipt Series Control
• Sequential Numbering
• X Reading
• Z Reading
• VAT Reports
• Refund and Void Monitoring

15. OFFLINE MODE & SYNCHRONIZATION
Description

Allows the POS system to continue operating during internet outages and automatically synchronize data once connectivity is restored.

Features

• SQLite Local Database
• Offline Sales Processing
• Offline Product Lookup
• Offline Customer Lookup
• Automatic Synchronization
• Conflict Resolution

16. SYSTEM SETTINGS
Description

Provides centralized configuration of business information, taxes, numbering, receipts, terminals, and application behavior.

Features

• Business Information Setup
• Receipt Customization
• Tax Configuration
• Number Series Configuration
• Payment Method Configuration
• Branch Configuration
• Terminal Configuration
